# Components

## Active Components

- **Hero.jsx** - Hero section with call-to-action buttons and statistics
- **ImageGallery.jsx** - Main food gallery with category filters and grid display
- **ProductDetail.jsx** - Individual product detail page
- **CartPage.jsx** - Shopping cart page (in pages folder)

## Unused/Alternative Components

- **FoodList.jsx** - Alternative food listing component (not currently used in routes). This appears to be an older implementation with different UI/UX. Kept for reference or future use.
- **AdminPanel.jsx** - Admin panel for creating food items (not currently routed). Can be added to routes if admin functionality is needed.
